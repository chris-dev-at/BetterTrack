Cookie note

Version 1.0 — effective 15 July 2026. Deutsche Fassung

BetterTrack sets exactly two first-party cookies, both strictly functional and both set only in response to something you do. There are no third-party, advertising or analytics cookies — which is why there is no cookie banner.

Session cookie (bt_sid)

Keeps you signed in to the web app. Set when you log in. By default it is a browser-session cookie that expires when your browser session ends; if you choose "stay signed in", it persists for up to 30 days. It is httpOnly and secure — no script on the page can read it.

Remembered-device cookie (bt_rdid)

Set only if you use "remember this device" (available with PIN quick re-auth). It stores an opaque device ID so the login page can offer "Continue as you" and a quick PIN sign-in instead of full credentials. It lives for up to about 400 days and is refreshed each time you use it. Choosing "Use another account" on the login page removes it immediately.

Local storage

The web app also keeps a few purely client-side entries in your browser's local storage — your language choice and, on a remembered device, the display record for the account chooser (username and avatar). These never leave your browser and are not tracking.

Clearing them

Log out to end your session, use "Use another account" to clear the remembered device, or delete cookies and site data for bettertrack.at in your browser settings at any time.

How cookie data is processed is described in the privacy policy.
